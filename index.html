ImageMagick API

Modify images from anywhere via a super simple API

View on GitHub Download .zip Download .tar.gz

ImageMagicAPI.com is not associated with ImageMagick Studio LLC - the makers of the ImageMagick tools - in any way

Accepts multiple formats

Input files can be specified as base64 encoded strings or as an external URL

Output

every API call returns the same output format:

{ "id": "http://www...." }

Common tasks are wrapped in helpers

/resize

Arguments
name type description required?
input base64 encoded string OR URI the input file Y
width integer the output width you must specify either width or height (or both)
height integer the output height see width
quality decimal output quality N
srcFormat string format of input file (e.g. JPG) N
format string format of output file (e.g. PNG) N
progressive boolean N
strip boolean N
filter string N
sharpening decimal N
Example
POST /resize HTTP/1.1
Accept: application/json
Content-Type: application/json
{
    "width": "100",
    "height": "100",
    "quality": 0.8,
    "srcFormat": "jpg",
    "format": "png",
    "progressive": false,
    "strip": true,
    "filter": "Lagrange",
    "sharpening": 0.2,
    "input": "iVBORw0KGgoAAAANSUhEUgAAAIAAAACACAYAAADDPmHLAAAAAXNSR0IArs4c6QAAAAlwSFlzAAALEwAACxMBAJqcGA"
}

/crop

Arguments
name type description required?
input base64 encoded string OR URI the input file Y
width integer the output width you must specify either width or height (or both)
height integer the output height see width
quality decimal output quality N
srcFormat string format of input file (e.g. JPG) N
format string format of output file (e.g. PNG) N
progressive boolean N
strip boolean N
filter string N
sharpening decimal N
gravity string one of {NorthWest, North, NorthEast, West, Center, East, SouthWest, South, SouthEast} N (defaults to Center)
Example
POST /crop HTTP/1.1
Accept: application/json
Content-Type: application/json
{
    "width": "100",
    "height": "100",
    "quality": 0.8,
    "srcFormat": "jpg",
    "format": "png",
    "progressive": false,
    "strip": true,
    "filter": "Lagrange",
    "sharpening": 0.2,
    "input": "iVBORw0KGgoAAAANSUhEUgAAAIAAAACACAYAAADDPmHLAAAAAXNSR0IArs4c6QAAAAlwSFlzAAALEwAACxMBAJqcGA"
}

Direct access to ImageMagick via HTTP

/convert

Arguments
name type description required?
input base64 encoded string OR URI the input file Y
args array arguments to pass (see http://imagemagick.org/script/command-line-processing.php#option) N
Example
POST / HTTP/1.1
Accept: application/json
Content-Type: application/json
{
    "input": "iVBORw0KGgoAAAANSUhEUgAAAIAAAACACAYAAADDPmHLAAAAAXNSR0IArs4c6QAAAAlwSFlzAAALEwAACxMBAJqcGA",
    "args": ['-resize', '25x120']
}
