Accepts multiple formats

All endpoints accept application/json or application/x-www-form-urlencoded. When posting as application/json input files can be specified as base64 encoded or as an external URL. When posting as application/x-www-form-urlencoded input files are posted as file fields strings representing external URLs.

Common tasks are wrapped in helpers

Direct access to ImageMagick via HTTP

Access ImageMagick command line tools simply:

montage -background '#336699' -geometry +4+4 rose.jpg red-ball.png montage.jpg

is recreated like:

POST /motage HTTP/1.1
Accept: application/json
Content-Length: 133
Content-Type: application/json

{
    "args": "-background #336699 -geometry +4+4",
    "input": [
        "iVBORw0KGgo...",
        "NSUhEUgAAAI...",
        "ADDPmHLEUgA..."
    ]
}
